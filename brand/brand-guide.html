Type: Inter (or the system UI font) for everything; a monospace for hashes, file counts, and status lines — it reinforces the “engineering / proof” story.

Usage

Do

  • Keep clear space around the mark equal to the shield’s width.
  • Use Signal Gold for one primary action per screen.
  • Reserve Verified Green strictly for confirmed / passed states.
  • Default to dark surfaces — it’s a studio tool.

Don’t

  • Recolor the mark outside the approved palette.
  • Stretch, rotate, or add drop shadows to the lockup.
  • Use red for anything other than risk / errors.
  • Place the full-color logo on busy photos.
